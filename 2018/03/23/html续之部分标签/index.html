html续之部分标签

Posted on 2018-03-23   |  

css布局

———
css布局由横向布局和纵向布局构成。横向布局中常常有纵向布局。

html标签

iframe标签（用的很少）

iframe可以使用css控制外观

a标签（get请求）

点击a标签后显示在iframe中
设置iframe的边框，一般设为0

frameborder=“0”

_blank在新开页面打开 _self在自身页面打开 _parent在父级窗口打开 _top在最顶级页面打开
a标签的下载属性

<a href="http://qq.com" download>下载</a>

a标签添加参数

<a href="?name=yzb" >qqq</a>

a标签添加javascript伪代码
a标签点击不跳转

<a href="javascript:;" >qqq</a>

a标签点击后跳转到页面上部

<a href=“#” >qqq</a>

a标签跳转到自身，刷新页面

<a href="" >qqq</a>

form标签(post请求)

失去submit提交将无法传输数据
file协议不支持post
get默认将提交内容放到查询参数里，也即地址栏里。不会放入到第四部分
post也可以将内容放入到查询参数里
<form action="users?xxx=1” method="post">即可
form 与a标签一样使用target，也可target到iframe中

input标签

如果一个form里只有一个按钮，<button>button</button>，执行submit功能。如果写了type=button，则失去这种功能
<input type="button" value="button">没有提交功能
<input type=“submit” value="button">可以提交
<input type="checkbox" name="yyy"><label for="yyy">loveme</label>

与下面代码功能一样，但下面代码更简洁，所以，用下面的代码了

<label><input type="checkbox" name="zzz">hateme</label>

input里的name还是要写的，传送数据时需要

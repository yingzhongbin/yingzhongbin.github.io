##重要内容

字体默认的 line height 由字体设计师决定

设置的行高可能会变动

CSS 中 width 的默认值auto

div的高度有其内部文档流元素的高度总和决定

文档流：文档内元素的流动方向。inline从左往右流动，block从上往下流动

font-size的大小指的是字体中的字从最上方到最下方的高度。而包含字的span的高度与字体的建议行高有关，当设置line-height行高小于字体的建议行高，会有bug。大于时可垂直居中。

不同字体放在一起将进行基线对齐

一个元素被fixed定位后， 它的宽度不会往外扩展，而是往内缩。解决问题的方法为宽度设为width:100%;但如果有padding的话，这样设置会出bug，解决方法是设置一个子元素，在子元素里设置padding。

使用svg进行垂直居中时，设置vertical-align: top;

给display:inline元素设置宽高是无效的，设置margin-left,margin-right有效，margin-top，margin-bottom无效

设置了display:inline-block;内容下方会有一个空隙，设置vertical-align:top;可以去除空隙
